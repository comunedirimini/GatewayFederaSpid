Gateway SPID/FEDERA

Il gateway permette di fruire della funzionalità autenticazione SPID/FEDERA, da parte degli applicativi interni dell’ente, rendendo trasparente la complessità dell'interfacciamento.

Il gateway di autenticazione permette di implementare un livello di astrazione fra le applicazioni (SP) e l'implementazione dell'interfacciamento con il provider di identità digitali (IdP) e si occuperà di gestire tutte le interazioni SAML con l'Idp e di restituire all' SP solo i dati dell'eventuale utente autenticato.

Il gateway è stato realizzato dal Comune di Rimini ed il codice sorgente è disponibile al presente collegamento.

Istanze Digitali

La piattaforma ISTANZE DIGITALE fornisce all’ente il servizio di creazione moduli parametrici per l’invio di istanze generiche create dagli uffici per i vari procedimenti come ad esempio Richiesta Numeri Civici, Richiesta di contributo centri estivi, ecc.

ISTANZE DIGITALE è un software creato dal Comune di Rimini premiato a Forum PA 2017 di Roma - Link alla notizia
